# Cadastro-de-Filmes
## Instruções
### Faça uma página de cadastramento de filmes onde o usuario possa salvar um filme e suas informações como:
 * Nome;
 * Data;
 * Descrição;
 * Diretor;
 * Categoria;
   ### Liste as informações utilizando algum elemento HTML.
## Tecnologias utilizadas:
- HTML;
- CSS;
- JavaScript.
